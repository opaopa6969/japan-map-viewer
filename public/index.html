🗾 japan-map-viewer

日本地図ビューア — 住所点群・コロプレス・自治体統廃合タイムライン。ライブラリ本体は /js/mapcore/(純ESM)。

  • レイヤー/演出デモ — 路線network・電車movers・人口柱extrusion・PiP・focusOn・wipe・heading-up (issue #1 の新API)
  • 住所マップ GL — MapLibre + deck.gl。点群/ヘックス/ヒート+コロプレス (推奨・basemap 生成不要)
  • 住所マップ 2D — canvas 直描き (要 npm run fetch:basemap)
  • 自治体マップ — 市区町村コロプレス
  • 統廃合タイムライン — 自治体統廃合の再生

住所点群は既定で同梱サンプル。VACANT_SERVICE_URL を設定すると vacant-service の /api/errorAddresses/geo をプロキシして実データを表示する。
